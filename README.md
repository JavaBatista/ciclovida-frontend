## Execução

Primeiro execute o backend na pasta backend.

```sh
java -jar ciclovida-backend-0.0.1-SNAPSHOT.jar
```

Instale as dependências.

```sh
yarn install
```

Execute o projeto.

```sh
yarn start
```
